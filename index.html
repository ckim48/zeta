    About hygiene heroes

    Since 2014, a team of UC Berkeley students under Haas Professor David Levine has developed a curriculum to teach preventative health in underprivileged regions. In prototyping our materials, we learned the most effective way to change behavior is through interactive styles of learning, such as stories, games, and songs. We now work with students and faculty at Cal Poly San Luis Obispo as well as local experts from Asia, Africa, and South America to improve our materials.

    We envision a world where children do not suffer from easily preventable diseases. We aim to realize that goal through interactive, educational lessons designed. Our materials are based on current medical and public health knowledge and checked with health professionals and educators in each nation where we work.

    What we do

    We created a point-based browser game (in HTML/CSS/JS) to teach young children about hygiene. The user can gain points whenever they destroy a hazard with a gun they can control.

    The character in the game can move around and try to dodge the many hazards moving around such as dirty hands. The goal is to accumulate the most points possible.

    Highlights

    Wireframe

    Wireframe is a blueprint showing the composition of the screen before game development is designed in advance so that the interaction of each element, such as game assets, UI, and characters, can be quickly referenced on the game design blueprint.

    HTML/CSS/JavaScript

    HTML, CSS, JavaScript are programming languages for web developments. HTML is the structure of a website, CSS styles for color, size, and JavaScript is custom Used to insert features.

    Object-Origend Programming

    Object Oriented Programming is a programming technique that divides and manages code into 'objects'. All objects used in the game, such as characters, obstacles, and soap, were classified into these objects and the code was written.
